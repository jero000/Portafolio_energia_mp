## 🛠️ Personalización

### Información de Contacto

Edita los siguientes datos en `index.html` y `script.js`:

1. **Instagram**: Actualiza el enlace en la sección de contacto
2. **WhatsApp**: En `script.js`, línea 91, reemplaza:
   ```javascript
   const whatsappNumber = '573XXXXXXXXX';
   ```
   Con tu número real (código de país sin +)

3. **Email**: Actualiza en la sección de contacto
4. **Ubicación**: Modifica según tu ubicación específica

### Colores y Branding

Los colores principales se definen en `styles.css` en las variables CSS:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    /* ... más colores */
}
```

### Agregar Imágenes

Para agregar tu foto o logo:

1. Guarda la imagen en el mismo directorio
2. En `index.html`, reemplaza la sección `.image-placeholder` con:
   ```html
   <img src="tu-imagen.jpg" alt="energia_mp">
   ```

### Testimonios

Personaliza los testimonios en `index.html` en la sección `testimonials` con datos reales de tus clientes.

## 🌐 Despliegue

### Opción 1: GitHub Pages (Gratis)

1. Crea un repositorio en GitHub
2. Sube todos los archivos
3. Ve a Settings > Pages
4. Selecciona la rama main como fuente
5. Tu sitio estará en: `https://tu-usuario.github.io/nombre-repo`

### Opción 2: Netlify (Gratis)

1. Arrastra la carpeta completa a [Netlify Drop](https://app.netlify.com/drop)
2. Tu sitio estará disponible en minutos

### Opción 3: Vercel (Gratis)

1. Crea una cuenta en [Vercel](https://vercel.com)
2. Importa tu proyecto
3. Despliega automáticamente
